### ```ImageProcessing.java```

Command line menu-driven program to perform certain operations on an image and generate the output file

Operations include:
 1. Negative
 2. Change Brightness
 3. Filter
    1. Red
    2. Green
    3. Blue
    4. Yellow
    5. Magenta
    6. Orange
    7. Cyan
 4. Monochromatic
    1. Grayscale
    2. Antique
 5. Flip
    1. Horizontal
    2. Vertical
 6. Rotate
 7. Crop
 8. Blur
 9. Sketch
 
 Output images are stored in a folder named ```Output Files```
